Per Command

To use Crate.io on a per command basis you can use pip's -i option to tell it to use http://simple.crate.io/ as it's index.

For Example if you wanted to install lxml from Crate.io you would simple type pip install -i http://simple.crate.io/ lxml

Per Requirements File

Pip also allows you to setup a custom index per requirements file, doing this is just as simple as per command, but it saves you from having to type it out each time.

To enable Crate.io for a particular requirements file, just edit the file and add -i http://simple.crate.io/ as the first line of the file. Save it and then run pip install -r requirements.txt

Globally

You've tried it out per command, set up some of your requirements files to use it, now you're ready to go all out and Globally switch to Crate.io

Luckily Pip makes this easy for us again and you can set up your global index url in pip.conf.

If you are on Unix or OSX your pip.conf is located at $HOME/.pip/pip.conf while on windows it's located at %HOME%\pip\pip.ini

Once you have your pip.conf file open and ready to edit, just add (or edit!) your file so it includes a section like 

    [global]
    index-url = http://simple.crate.io/
